HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Hierarchical heading tags and alt text should be used for screen assisted users.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the outside of an element.
  • The border attribute indicates the pixel thickness of the border.
  • From the outside in, the CSS box model follows the order: margin, border, padding, content.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git pull origin branch-name: pulls the branch from GitHub to the local repo.
  • git push origin branch-name: pushes the branch from the local repo to GitHub.

JavaScript

  • A variable is a named container that allows us to store date in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Data types in JavaScript variables do not need to be explicitly typed.
